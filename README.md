**Student Result Management System** – Ваш надежный помощник в учете успеваемости!

**Что это такое?**
Приложение для автоматизации процесса учета и расчета результатов студентов. Позволяет быстро вычислять итоговые баллы, проценты успеваемости и сохранять данные для последующего просмотра.

**Основные возможности:**
- Авторизация для безопасного доступа
- Ввод оценок по трем предметам
- Автоматический расчет общего балла и процента успеваемости
- Определение результата (сдал/не сдал)
- Сохранение результатов в файл
- Просмотр всех сохраненных результатов

**Для кого подойдет:**
- Для преподавателей и классных руководителей
- Для администраторов учебных заведений
- Для студентов, изучающих программирование

**Преимущества:**
- Простая и понятная система авторизации
- Быстрый расчет успеваемости
- Сохранение данных в текстовый файл
- Удобный просмотр всех результатов

Упростите процесс оценки знаний с нашим приложением!
